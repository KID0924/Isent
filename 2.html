每日詳細行程

D1 5/19 龍頭巖與奇幻夜市

今日踩點

桃園機場出發 → 龍頭巖 (看神話石頭) → 韓國東邊松堂童話村 (濟州最大星巴克在這!) → 神奇之路 (體驗錯覺奇幻山坡) → 漢拏樹木園露天夜市 (綜藝玩很大推薦，吃飽喝足)

早機上輕食 (水要自己準備)

中濟州豚骨家 (馬鈴薯燉豬骨湯 + 季節小菜 + 米飯)

晚濟州蔘雞莊 (韓式綠茶人蔘雞 + 泡菜)

D2 5/20 瀑布與治癒森林

今日踩點

正房瀑布 (亞洲唯一直接落海瀑布) → 西歸浦治癒森林 (必穿運動鞋!) → 西歸浦每日偶來市場 (當地人也愛逛的傳統市場) → 山房山 → 城邑民俗村 (看韓國茅草屋，品嚐招待的蜂蜜茶/五味子茶)

早飯店內早餐

中一品莊 (韓式炸醬麵 + 糖醋肉四人一份 + 泡菜)

晚北村家 (豆腐鍋 + 營養石鍋飯 + 烤魚 + 季節小菜)

D3 5/21 網美小火車與商圈

今日踩點

Ecoland 歐洲英式森林小火車 → 山茶花之丘 (春季賞繡球花等花卉) → 梨湖紅白馬小燈塔 (IG熱門打卡點) → 蓮洞時尚商圈 (繁華區買爆化妝品服飾)

早飯店內早餐

中濟州私廚 (黑毛豬 + 豬排 + 燒烤吃到飽！每桌送鮑魚+鮮蝦)

晚發放 ₩6000 韓幣(請自己去商圈覓食)

D4 5/22 文化體驗與東門夜市

今日踩點

觀音寺 (非常律師禹英禑拍攝地) → 精裝彩妝坊 (跟上時代潮流買起來) → 韓國文化體驗 (水果大福 DIY + 穿韓服拍美照 + 海苔博物館) → 東門傳統市場/夜市 (食尚玩家推薦美食探索)

早飯店內早餐

中漁夫廚房 (超級海鮮鍋：活章魚、鮑魚、蟹、貝類蝦 + 煎餅 + 烤魚)

晚發放 ₩6000 韓幣(夜市自己買起來吃)
